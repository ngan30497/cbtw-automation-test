# CBTW Automation Test

## Project Description
This project aims to provide an automated testing framework for the CBTW application, enhancing reliability and speeding up the testing process.

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/ngan30497/cbtw-automation-test.git
   cd cbtw-automation-test
   ```  
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Configure environment variables** as needed for your local setup.

## Structure
- `src/`: Contains source code for the automated tests.
- `tests/`: Contains test cases and scripts.
- `docs/`: Documentation for the project.
- `README.md`: Project overview and instructions.

## Running Tests
To execute the tests, run the following command in the terminal:
```bash
npm test
```
This will run all the defined test cases in the `tests/` directory.

### Test Results
Check the console output for the results of each test. For more detailed results, refer to the logs generated in the `logs/` folder.
